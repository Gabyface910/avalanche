# Avalanche
is a Raspberry Pi CLI app store (inspired by good 'ol pi-apps)

## Installing Avalanche
Installing Avalanche is as simple as running one command in your terminal! Run this to get started:
```bash
curl -fsSL https://raw.githubusercontent.com/Gabyface910/avalanche/refs/heads/main/install.sh | bash
```
It's as simple as that! Here's an app to get started with:
**En Croissant** (Chess GUI) - How to install
```bash
sudo avl install en-croissant
```
